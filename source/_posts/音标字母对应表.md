掌握了[音标](https://zhuanlan.zhihu.com/p/35702220)和其对应的字母（组合），就可以根据读音来拼写单词了。

<!-- more -->

### 元音(20 个)

| 元音   | 字母（组合）                         |
| ------ | ------------------------------------ |
| `[i:]` | `ee` `ea` `e` `ie`                   |
| `[i]`  | `i` `y` `e`                          |
| `[æ]`  | `a`                                  |
| `[e]`  | `ea` `e` `a`                         |
| `[ə:]` | `ir` `ur` `ear` `er` `or`            |
| `[ə]`  | `er` `or` `ou` `ar` `o` `a` `e` `u`  |
| `[a:]` | `ar` `a`                             |
| `[۸]`  | `u` `or` `ou` `oo`                   |
| `[כ:]` | `al` `or` `au` `our` `ar` `aw`(英音) |
| `[כ]`  | `o` `a` `aw`(美音)                   |
| `[u:]` | `oo` `o` `u`                         |
| `[u]`  | `oo` `ou` `o` `u`                    |
| `[ei]` | `a` `ay` `ea` `ai` `ey`              |
| `[ai]` | `i` `y`                              |
| `[au]` | `ou` `ow`                            |
| `[əu]` | `o` `ow` `oa`                        |
| `[כi]` | `oy` `oi`                            |
| `[iə]` | `eer` `ea` `ear` `er` `ier`          |
| `[eə]` | `ear` `air`                          |
| `[uə]` | `our` `oor`                          |

| ------ | ------------------ |
| `[k]`  | `k` `c`            |
| `[ŋ]`  | `ng` `n`           |
| `[j]`  | `y`                |
| `[f]`  | `f` `gh` `ph`      |
