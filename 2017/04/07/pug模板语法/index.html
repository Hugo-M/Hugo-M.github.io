模板继承

关键词 描述
block 占位符，通过其实现子模板的追加和替换
extends 继承，子模板继承父模板中的代码
append 在子模板中向后追加内容
prepend 在子模板中向前追加内容
include 引入文件，并将其中的代码放到模板中
